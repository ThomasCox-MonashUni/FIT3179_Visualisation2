Programming Across the Globe

A visualisation showing how the programming languages of today look like around the world.

JavaScript is currently the most commonly used programming language in most countries, used by 11.9% of programmers worldwide.

JavaScript has also been the most commonly used programming language worldwide over the past 6 years, followed by Python and TypeScript. Java has started to wane in popularity, having declined 3.75% since 2017.

Men make up the majority of programmers, making up 95% of programmers. The least number of programmers are Trans and Non-Binary, making up only 1% of the total.
